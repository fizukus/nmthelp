Конвертер балів тестування у шкалу НМТ

Конвертуйте бали тестування за системою НМТ для різних предметів

Як це працює:

  1. Завантажте Excel-файл (.xls або .xlsx) з результатами тестування
  2. Оберіть предмет для конвертації балів
  3. Вкажіть назви стовпців у вашому файлі
  4. Отримайте готовий файл з балами НМТ

Формат файлу: Excel (.xls, .xlsx)

Стовпці у файлі: ПІБ учнів та бали тестування

Якщо файл не завантажується:

  1. Перевірте формат файлу (має бути .xls або .xlsx)
  2. Спробуйте переіменувати файл без кирилиці
  3. Переконайтеся, що файл не відкритий у Excel
  4. Якщо не працює - відкрийте сторінку через локальний сервер (див. інструкцію нижче)
